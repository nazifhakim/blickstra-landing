# Brand Assets — /public/brand/

Drop your Blickstra logo files here:

| File | Description |
|------|-------------|
| `blickstra-logo-light.svg` | Full logo — for dark backgrounds |
| `blickstra-logo-dark.svg` | Full logo — for light backgrounds |
| `blickstra-logo-compact.svg` | Compact logo — light backgrounds |
| `blickstra-logo-compact-dark.svg` | Compact logo — dark backgrounds |
| `blickstra-mark.svg` | B mark only (icon/favicon source) |
| `blickstra-wordmark.svg` | Text wordmark only |

Also place at `/public/`:
- `favicon.svg` — replace existing (use blickstra-mark.svg at 32×32)
- `apple-touch-icon.png` — export mark at 180×180px
- `og.png` — 1200×630px Open Graph image for social sharing
