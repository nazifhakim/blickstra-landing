COLOUR USAGE RULE — EMERALD SCARCITY

Emerald #3ecf8e is the sole chromatic accent in the brand system. It must appear once per viewport — either as a filled CTA button, the logo mark, or an active nav state. Never both simultaneously. This scarcity is what gives it visual power.

CORRECT USE

Logo mark on page header — no other emerald element visible on same scroll position

INCORRECT USE

Emerald logo + emerald CTA button + emerald active nav all visible at once

08

Do's & Don'ts

DO

  • Use all-caps BLICKSTRA in the wordmark
  • Use near-black text (#0d1a12) on the emerald button
  • Keep emerald as the single chromatic accent per viewport
  • Use Inter weight 500 for all display headings
  • Maintain 12px (lg) radius on the logo mark
  • Use 6px (sm) radius on all buttons
  • Use white canvas (#ffffff) as the primary background
  • Apply negative letter-spacing on display type

DON'T

  • Use white text on the emerald CTA button
  • Stretch, rotate, or distort the logo mark
  • Use the logo on busy photographic backgrounds
  • Introduce additional accent colours (no blue, red, orange as brand)
  • Use font weight 600 or 700 anywhere in the UI
  • Use pill-shaped buttons (brand radius is square-ish 6px)
  • Add gradients or atmospheric backgrounds behind the hero
  • Place the wordmark below minimum size (20px height)
